Binding:-
  Association of method definition to the method call is known as binding.
  
two types:-
    1.Dynamic binding
	2.static binding(Early binding)





Dynamic binding(late binding):-
    1.parent class object child class memory refer
    2.child class-only override method are acceptable not variable 
    3.runtime
	4.compiler is not able to resolve at compile time  by compiler Dynamic or late Binding
   
    Eg:Overriding
	
     Parent p = new Child();//p refer parent class
  
  

Static binding(Early binding):-
    1.compile time
	2.resolved at compile time by compiler is known as static or early binding
 
    Eg:overloaded
	
	parent p = new parent();
